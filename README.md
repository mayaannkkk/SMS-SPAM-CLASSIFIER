# 📩 SMS Spam Classifier

A simple and efficient **Machine Learning web app** that classifies SMS messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing techniques.

---

## 🚀 Live Demo

👉 https://sms-spam-classifier-gzvzjctwxy2bja9gngc5sw.streamlit.app/

---

## 📌 About the Project

This project uses a trained machine learning model to detect whether a given SMS message is spam or not. It applies basic text preprocessing techniques and transforms input text into numerical vectors before making predictions.

The app is built using **Streamlit**, making it interactive and easy to use.

---

## ⚙️ Features

* 📥 Input any SMS message
* 🔍 Automatic text preprocessing
* 🤖 Spam/Ham prediction using ML model
* ⚡ Fast and lightweight interface

---

## 🧠 Tech Stack

* **Python**
* **Streamlit**
* **Scikit-learn**
* **Pandas & NumPy**
* **Natural Language Processing (NLP)**

---

## 🛠️ How It Works

1. User enters a message
2. Text is cleaned and processed
3. Converted into vectors using TF-IDF
4. Model predicts whether it's spam or ham
5. Result is displayed instantly
